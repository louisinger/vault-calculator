# Vault Calculator

Lock your Liquid assets using a simple Ionio contract & Marina wallet.

![GitHub Pages](https://github.com/vulpemventures/marina-api-test-app/actions/workflows/gh-pages.yml/badge.svg)

## Development

### Install dependencies

```bash
yarn install
```

### Run the app

```bash
yarn dev
```

### Build the app

```bash
yarn build
```
