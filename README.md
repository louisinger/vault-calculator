# Vault Calculator

https://louisinger.github.io/vault-calculator/

Lock your Liquid assets using a simple Ionio contract & Marina wallet.

## Development

### Install dependencies

```bash
yarn install
```

### Run the app

```bash
yarn dev
```

### Build the app

```bash
yarn build
```
